
# DAILY CLAIM DROPAIR




## Run Locally

Clone the project

```bash
  git clone https://github.com/xthample/dropair
```

Go to the project directory

```bash
  cd dropair
```

Install dependencies

```bash
npm install axios
npm install cli-spinner
npm install prompt-sync
npm install dotenv
```

Start the server

```bash
  node daily.js
```
